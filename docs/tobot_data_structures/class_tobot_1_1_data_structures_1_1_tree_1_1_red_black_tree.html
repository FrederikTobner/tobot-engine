Detailed Description

template<typename T>
class Tobot::DataStructures::Tree::RedBlackTree< T >

A RedBlackTree is a self-balancing binary search tree.

Template Parameters
TThe type of the value stored in the tree

It is a binary search tree with one extra bit of storage per node: its color, which can be either red or black. These colors are used to ensure the tree remains approximately balanced during insertions and deletions.
